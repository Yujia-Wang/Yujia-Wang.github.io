Problem

Intelligent control technology has developed increasingly these years. And people love to try new technology that can make life easier. However, there are still some problems when considering smart home control system, which make them not pervasive to everyone.

  • People hate to download too many apps on their phones.
  • Intelligent appliances can be very expensive.

Solution

Here we present WeHome, an interactive home control system in WeChat that helps user easily control their electronic appliances. This system is based on WeChat, the most popular social media in China, and it's suitable to most appliances. It mainly has the following functions:

  • Gather environment information, such as temperature.
  • Turn on/off appliances.
  • Adjust the brightness of light.
  • set up timer for appliances.

Process

This project mainly focused on the development of interactive technology. Based on the existing technology, we expanded its functions and made a future plan for it.

Prototype

After determining the technology and functions, we started to build the prototype system. This prototype contains two parts: pyhsical prototype (ZigBee development) and platform construction (WeChat development).

To build a pyhsical prototype, we decided to reform socket so that normal appliances can be upgraded into intelligent appliances. We implemented several terminal nodes connecting appliances and a coordinator node connecting server. Each terminal node controls one appliance. Evaluation

After we finished the prototype, we recruited several users to go through our prototype. By conducting user testing, we were able to get their impressions and identify issues on our design.

Control Modes

Voice control, text input control, user interface control

Participants

Each participant tested all three controlmodes in random order

Metrics Measured

Learnability, efficiency, error and satisfication

For each control mode, we set up exactly the same tasks for participants to perform, including:

  • Get the environment temperature.
  • Set the bedroom lamp to turn off in 2 minutes later.
  • Turn on fan.

Results

The graphs below are an overall evaluation on our design based on the results from user testing. We discovered several things related to our design:

  • Our system is easy to learn in all three control modes, the instruction is very clear for users to understand.
  • Poor internet can impact on our system a lot, slow internet transmission leads to slow feedback, which minimizes user experience.
  • User interface control is the most intuitive, compared with other two control modes.
  • Users expect more functions in this system, for example, fire alarm.

Recommendations

Based on the feedback we received, we were able to identify issues existing in our design and provide the following recommendations for future work:

  • Implement more environment sensors, for example, smoke and luminous intensity.
  • Realize the functionality of automatically adjusting appliances according to environment data, for example, turn on or off fan according to temperature.
  • Optimize the design of our intelligent socket.
  • Implement the functionality of privacy control, use account and password to make multiple users control their own appliances seperately but in one official WeChat account.
  • Optimize the user interface, provide feedback when users use certain function.
  • Implement the functionality of adding or deleting appliances by user self, provide users more freedom to control their appliances.
